== Description ==

Plugin for sharing wordpress posts into WPDelicious.

== Installation ==

1. Upload `wp-delicious` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go To Themes - Sidebar And add `Latest posts in wpdelicious` to widget.

== Frequently Asked Questions ==

= How does it work? =
Sent url Posts to delicious wordpress (http://wpdelicio.us/).
Show Latest Post WPDelicio.us in Widget.
